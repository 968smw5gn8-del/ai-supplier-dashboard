AI Supplier Valuation Grid

Mobile‑first dashboard for intrinsic value modeling of AI infrastructure suppliers. Live prices refresh every 60 seconds via Yahoo Finance; you can tweak assumptions per company and see buy/sell/hold/wait signals update in real time.

Live data: Yahoo Finance (60s interval)

This page is a client‑side single‑page app designed for GitHub Pages deployment. All modeling is local to your browser. Live data uses public Yahoo Finance endpoints via fetch; for production, consider adding a small proxy/API layer to handle rate limits and CORS. Nothing here is investment advice—just a sandbox for thinking clearly.
